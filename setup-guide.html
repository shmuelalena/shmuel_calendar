Google Cloud Console
כבר יש לך פרויקט — צריך רק להפעיל Calendar API ולצור Client ID.
↗ פתח Google Cloud Console
1
בחר את הפרויקט שלך מהתפריט הנפתח בראש הדף
2
בתפריט הצדדי: APIs & Services ← Library
3
חפש Google Calendar API ← לחץ עליה ← לחץ Enable
4
אם כבר מופעלת (כתוב "Manage") — אין צורך לעשות כלום, המשך הלאה ✓
💡 אם זה הפרויקט הראשון שלך ב-Google Cloud, תצטרך גם להגדיר OAuth Consent Screen — נעבור על זה בשלב הבא.
🔑
יצירת OAuth Client ID
זה המפתח שמאפשר לאפליקציה שלך להתחבר ל-Google בשמך.
1
לך ל: APIs & Services → Credentials
2
לחץ + Create Credentials → OAuth Client ID
3
Application type: בחר Web Application
4
Name: Shmuel Calendar
5
Authorized JavaScript origins — לחץ Add URI והכנס את ה-URL של האפליקציה שלך.
למשל: https://shmuel-calendar.netlify.app
אם עדיין אין לך URL — הכנס http://localhost ועדכן אחרי הפריסה.
6
לחץ Create — יופיע חלון עם Client ID
נראה כך: 123456789-abcdefg.apps.googleusercontent.com
⚠️ OAuth Consent Screen — אם מבקש להגדיר, לחץ "Configure Consent Screen" → External → מלא שם האפליקציה → הוסף את האימייל שלך כ-Test User → Save.

פריסת האפליקציה
כדי ש-Google Calendar יעבוד, האפליקציה חייבת לרוץ על HTTPS. Netlify מספק את זה חינם תוך דקה.

2
בדף הראשי לחץ "Add new site → Deploy manually"
3
גרור את תיקיית shmuel-calendar מה-ZIP שהורדת ← שחרר
4
תוך 30 שניות תקבל URL! העתק אותו למטה.
1

2
העלה את כל הקבצים מה-ZIP → Commit changes
3
Settings → Pages → Source: Deploy from branch → main → Save
4
ה-URL יהיה: https://USERNAME.github.io/shmuel-calendar

סנכרון מכללת SCE
מערכת השעות של SCE אפשר לסנכרן דרך Google Calendar — זה הכי נוח!
📅 שיטה 1: ייצוא iCal (מומלץ) קל ביותר
1
כנס ל-פורטל הסטודנטים של SCE (my.sce.ac.il)
2
מצא את לוח השנה / מערכת שעות → ייצוא → iCal / .ics
3
ב-Google Calendar: Other Calendars → Import → העלה את קובץ ה-.ics
4
✅ כל השיעורים יופיעו ביומן שלך — ובאפליקציה אוטומטית!
🔗 שיטה 2: Subscribe URL
1
אם SCE מספקת Calendar URL — העתק אותו
2
Google Calendar: Other Calendars → From URL → הדבק → Add Calendar
מתעדכן אוטומטית כל שינוי במערכת השעות!
💡 אחרי הייבוא — כל השיעורים יסונכרנו אוטומטית ליומן החכם דרך ה-Google Calendar API שהגדרנו.
✅ הקובץ המוכן שלך
הדבק את הפרטים האלה בתוך האפליקציה (בהגדרות Google Calendar):
// פרטי ההגדרה שלך:
Client ID: [ הכנס בשלב 2 ]
App URL:   [ הכנס בשלב 3 ]

// הדבק את ה-Client ID בתוך האפליקציה
// בלשונית 🗓️ Google → שדה Client ID
